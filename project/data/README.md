# Данные проекта

В этой директории находятся данные, необходимые для обучения модели и работы сервиса кредитного скоринга.

## 1. Правила хранения данных

Согласно требованиям курса (`faq.md §2.1–2.3`) и политике приватности:

✅ **Допустимо хранить:**
- Инструкции по загрузке открытых датасетов.
- Небольшие примеры (sample) для проверки структуры.
- Обезличенные учебные выборки.

❌ **Запрещено хранить в репозитории:**
- Реальные персональные или конфиденциальные данные.
- Полные «сырые» файлы датасетов объёмом более 50 МБ.
- Файлы с секретами, токенами или учётными данными.

Для данного проекта исходные CSV-файлы исключены из контроля версий через `.gitignore`.

## 3. Описание датасета

| Параметр | Значение |
|----------|----------|
| **Название** | Give Me Some Credit |
| **Источник** | [Kaggle Competition](https://www.kaggle.com/c/GiveMeSomeCredit) |
| **Объём** | ~150 000 записей |
| **Признаки** | 10 числовых (финансовые показатели, история просрочек, демография) |
| **Целевая переменная** | `SeriousDlqin2yrs` ∈ {0, 1} |
| **Тип данных** | Открытый, полностью обезличенный, учебный |
| **Лицензия** | Разрешено использование в некоммерческих и образовательных целях |

Датасет не содержит персональных идентификаторов. Все значения представлены в агрегированном виде.

## 4. Инструкция по загрузке полного датасета

Для полноценного обучения необходимо скачать исходные файлы:

1. Перейдите на страницу соревнования: https://www.kaggle.com/c/GiveMeSomeCredit
2. Примите правила соревнования (`Rules` → `Accept`).
3. В разделе `Data` скачайте `cs-training.csv`.
4. Поместите файл в `data/raw/`.
5. Убедитесь, что файл не попал в коммит (правило `project/data/raw/*.csv` в `.gitignore`).
